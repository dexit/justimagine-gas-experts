# FEATURES.md - Core System Features

## 1. Multi-Channel Lead Pipeline
- **Edge-First**: Submissions processed at the edge via Vercel Edge Functions.
- **HubSpot Native**: Direct integration with HubSpot CRM for immediate lead tracking.
- **Custom Webhooks**: Extensible dispatch to Cloudflare Workers or REST endpoints.

## 2. Advanced Design System
- **OKLCH Colors**: High-fidelity color space for perfect dark mode transitions.
- **Glassmorphism**: Premium frosted-glass effects for navigation and panels.
- **Dynamic Bento Grids**: Interactive, animated grid layouts for services and features.
- **Micro-animations**: Targeted UI animations using `tw-animate-css` and native CSS keyframes.

## 3. SEO & Area Expansion
- **Dynamic Regional Routing**: Auto-generated landing pages for specific Warwickshire areas.
- **Sitemap Automation**: Intelligent 300+ URL sitemap generation.
- **Rich Snippets**: Built-in Schema.org support for Local Business and Services.

## 4. Performance & Reliability
- **Vercel Speed Insights**: Real-time monitoring of Core Web Vitals.
- **Image Optimization**: Automated generation of WebP and responsive image sizes.
- **Reliable Fallbacks**: System-font fallbacks for custom fonts to prevent FOUT.

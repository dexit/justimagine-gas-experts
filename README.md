# Just Imagine Gas Experts

A premium, high-performance web platform for Warwickshire's leading gas and heating engineering firm.

## 🚀 Built for Speed & Growth
- **Edge Architecture**: Sub-millisecond form processing via Vercel Edge.
- **Dynamic SEO**: 300+ landing pages generated for hyper-local Warwickshire targeting.
- **Modern Stack**: TanStack Start, Vite, Bun, and Tailwind CSS 4.

## ✨ Design Philosophy
- **Premium Visuals**: Glassmorphism, organic textures, and OKLCH color precision.
- **Micro-Interactions**: Subtle, staggered animations for high user engagement.
- **Bento Grid Layouts**: Clean, modular information architecture.

## 🛠️ Development
```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Generate sitemap
bun run generate:sitemap
```

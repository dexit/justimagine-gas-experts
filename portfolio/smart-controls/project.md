# Smart Heating Controls & Room Thermostats

A modern detached house had manual controls but lacked flexibility. We installed a full smart heating system with app-based control and learning capabilities.

## The Challenge

- **Old controls:** Basic boiler dial, no flexibility
- **Scheduling:** Manual adjustments every season
- **Zoning:** Whole house on/off, impossible to manage individual comfort
- **Efficiency:** Overheating rooms to keep one warm
- **Travel:** Away for week, couldn't turn down heating remotely

## Our Solution

**Smart Control System:**
1. **Nest Learning Thermostat** – Central unit with learning algorithm
2. **Wireless receiver** – Replaces old control, communicates via 868 MHz
3. **Smart valves** – Room thermostats in 3 zones (upstairs, downstairs, study)
4. **App control** – iOS + Android app for on-the-go adjustments
5. **Geofencing** – Automatic turn-off when home empty, turn-on when approaching
6. **Energy reports** – Weekly breakdown of usage by room/time

## Features Enabled

- **Room-by-room control:** Each zone set independently
- **Learning mode:** System learns heating patterns, suggests optimizations
- **Remote access:** Adjust from anywhere via smartphone
- **Scheduling:** Different settings for weekdays/weekends/holiday
- **Energy insights:** See which rooms use most energy, when peaks occur
- **Integration:** Works with other smart home systems (Google Home, Alexa)

## Results

- **Energy savings:** 20% reduction in winter gas consumption
- **Comfort:** Bedrooms now exactly at desired temperature
- **Convenience:** Never come home to cold house again
- **Cost:** £1,200 for full installation
- **Payback:** Saves ~£250/year, pays for itself in 5 years
